About This Application

📋 Application Overview

Album Browser is a single-page application (SPA) built with Angular that allows users to browse, view, edit, and delete albums fetched from a REST API. It demonstrates core Angular concepts including routing, HTTP communication, services, and reactive data handling with Observables.

✅ Features

  • Multi-page SPA with 6 configured routes
  • Service layer with 5 HTTP methods (GET, PUT, DELETE)
  • Album list with delete functionality
  • Album detail view with editable title
  • Photo gallery with responsive grid layout
  • Loading indicators and empty state handling

🔗 API Source

Data is powered by JSONPlaceholder — a free fake REST API for testing and prototyping.
